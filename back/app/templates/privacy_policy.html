Política de Privacidad

1. Qué datos guardamos

Recopilamos y almacenamos la siguiente información:

  • Datos de cuenta:
    • Nombre de usuario
    • Correo electrónico
    • Contraseña (almacenada de forma segura y encriptada)
    • Imagen de perfil
  • Datos de juego:
    • Historial de partidas
    • Estadísticas (victorias, derrotas, partidas jugadas)
  • Datos de seguridad:
    • Estado de autenticación de dos factores (2FA)
    • Tokens de acceso y actualización
  • Datos de interacción:
    • Lista de amigos
    • Lista de usuarios bloqueados
    • Historial de mensajes de chat

2. Por qué guardamos estos datos

Los datos se almacenan para:

  • Proporcionar y mantener tu cuenta de usuario
  • Gestionar tu progreso en el juego y estadísticas
  • Facilitar la interacción social entre usuarios
  • Garantizar la seguridad de tu cuenta
  • Mejorar la experiencia de usuario

3. Cómo protegemos tus datos

Implementamos las siguientes medidas de seguridad:

  • Encriptación de contraseñas usando algoritmos seguros
  • Autenticación de dos factores (2FA) opcional
  • Tokens JWT para sesiones seguras
  • Almacenamiento seguro de datos en bases de datos PostgreSQL
  • Protección contra accesos no autorizados
  • Integración con HashiCorp Vault para gestión segura de secretos

4. Tus derechos sobre tus datos

Como usuario, tienes derecho a:

  • Acceder a tus datos personales
  • Modificar tu información de perfil
  • Solicitar la eliminación de tu cuenta
  • Exportar tus datos

Para ejercer estos derechos, puedes:

  • Acceder a tu perfil para ver y modificar tus datos
  • Contactar al administrador del sistema
  • Utilizar la función de eliminación de cuenta disponible en tu perfil

5. Retención de datos

Los datos se mantienen mientras tu cuenta esté activa. Al eliminar tu cuenta:

  • Se eliminarán todos tus datos personales
  • Se mantendrán solo las estadísticas de juego anónimas
  • Se eliminará tu historial de mensajes y relaciones sociales

6. Contacto

Para cualquier consulta relacionada con la privacidad de tus datos, puedes contactar al administrador del sistema.
